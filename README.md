# David Iraya Kivishi Foundation Website

A professional foundation website for Hon. David Iraya Kivishi, Member of County Assembly (MCA) for Manda-Shivanga Ward, Malava Sub-County, Kakamega County, Kenya.

## Project Overview

This website serves as a community leadership platform showcasing the development work, initiatives, and community service of Hon. David Iraya Kivishi. The site is designed to be professional, trustworthy, and service-oriented while maintaining a warm, community-focused tone.

## Getting Started

### Local Development
1. Clone or download the project files
2. Navigate to the `website` directory
3. Start a local server:

```bash
# Using Python 3
python3 -m http.server 8000

# Using Node.js (if available)
npx serve .

# Using PHP (if available)
php -S localhost:8000
```

4. Open your browser and visit `http://localhost:8000`

### Browser Compatibility
- Chrome/Chromium (recommended)
- Firefox
- Safari
- Edge
- Mobile browsers
